# Time Tracking System (CODE: LW2)

## Functional Scope

The system is designed for employees working in teams on projects and tasks. It enables efficient tracking, management, and reporting of working time.

### Core functionalities:

**Employees can:**
- Log time spent on tasks
- View assigned tasks and projects
- Edit or delete their time entries
- Track their own work history

**Administrators can:**
- Create and manage projects and tasks
- Assign employees to tasks
- Monitor time logs in real-time
- Generate reports (detailed and summary)
- Manage users and teams

**System features:**
- Time logging with validation (no overlaps, correct durations)
- Filtering and sorting of time entries
- Period-based reporting (daily, weekly, monthly)
- Dashboard with aggregated statistics
